usage

  1. paste your list of locations in the box above. you can paste them in the following formats:
    • a list of (unshortened) google maps links, one link per line
    • the json output generated by slashp's "copy/paste geoguessr map data" script
  2. click extract links
  3. once the script is finished, it will generate a json output that you can import with slashp's script
  4. any location that couldn't be fixed will be removed and listed in the errors list
